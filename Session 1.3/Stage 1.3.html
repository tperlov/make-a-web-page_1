What is the World Wide Web?

The World Wide Web is comprised of billions of HTML pages. It contains every kind of file you can think of. The World Wide Web is comprised of your browser, which connects to the Internet by communicating with servers using a protocol called HTTP. Servers are just computers and HTTP is just a protocol that your browser uses to communicate with the server.

HTTP (HyperText Markup Language) -The main protocol of the web
Servers - Computers that host the files that make up the web
Internet - the world's largest computer network.
Browser - a program that runs on your computer to display files found on the web

Basics of HTML

HTML Tags

Tags are instructions on how your page should be structured and are obvious within the code as they are enclosed in "<>". Text after a tag is affected all the way through until a closing tag is hit "</ >". Below are some examples.

<b> is a tag that indicates anything enclosed should be bold.
<em> indicates italic text.
<p> is a tag which creates something like a paragraph structure.
<br> creates a line break. This tag is unique because it doesn't have any contents and therefore does not require a closing tag.

Attributes

Some tags have attributes, which in turn have values. For example

The <a> tag is called an anchor tag and it's how we add in links. The <a> tag has an href attribute which has a URL value. Between the opening tag and the closing tag is the text which appears hyperlinked. For example I can say, "Dear reader, go visit <a href="http://www.reddit.com">reddit<a>." and it will look simply show

"Dear reader, go visit reddit."

Another example uses the <img> tag to insert images. This tag has two attributes called src and alt. src has a URL value just like href meanwhile alt gives you a word description which is displayed when the image is not loaded and can be used for special settings such as those for the blind. For example, here is an <img src="http://my10online.com/wp-content/uploads/2011/09/4480604.jpg" alt="Puppies">. This tag is another example of a tag that doesn't have text contents and therefore doesn't require a closing tag.

Inline vs. Block Elements

Inline elements are tags that create inline (within the text) formatting.
Examples include <b>, <em>, <br>,and <img>. Another tag that falls into this category is <span> which is a container tag.

Block elements are elements that create a virtual box around its contents. Examples include <p> which creates a block around the paragraph, and <div>which creates a block around the contents and is the block counterpart to the inline element <span>.

Stage 1.2

Boxifying

Boxifying is apparently the technical term for separating elements on a page into compartments. Any page of content can be split up into smaller blocks and sub-blocks. Each of these blocks can be placed into a container like <div>. This skill can be practiced by either drawing boxes around each element or even by printing the page and cutting the blocks out. Even shapes like circles are really just blocks.

HTML Structure

Aside from boxifying, you can also think of an HTML document as having a tree like structure. You can see this structure by following the indentations along the left end of the HTML code. For example, many trees will start with a <body> and the next line will have an indented element. This element is the next branch of the tree. Multiple <div> sections can be either along the same branch or they can be two separate limbs of the tree.

Stage 1.3

What is CSS?

CSS stands for Cascading Style Sheets and allows to add style to the html structure. Style means anything like color, formatting, shapes, or location of elements on a page.

How to use CSS

When dealing with CSS it's important to consider your HTML structure. Giving yours <div>'s a class name makes it possible to attach a style to that specific div.
For example, given <div class="stuff">, you could add in a style by specifying in your CSS file the selector, an element type, or by using the class name followed by the attribute and the appropriate value.

div {
color: red;
}

or

.stuff {
color: red;
}

and this would make all the text in that <div> red in color. You can also insert the CSS styling right into your html sheet using the <style> tag.

The Box Model

Boxifying was a term discussed earlier in Stage 1.2 but when we think of a simple box we miss all the elements that make it possible to configure how the content in the box will sit with respect to the box or how other boxes on the page will sit with respect to each other or to the page. For better handling, boxes come with 4 component: content, padding, margin, and border. To make these components easy to understand let's compare it to a peice of paper on a desk.

Imagine that you have a peice of paper. The size of this paper is your border. On this paper you have some scribbles which we will call the content. The space between your scribbles and the edges of the paper is your padding and the space between the edges of your paper and the edge of your desk is the margin.

Reducing Repetition

It is important to avoid repetition. Not only does it create more reasable code but it reduces the amount of work required each time a change is needed. There are a couple ways to achieve this. The first is to use class names for multiple div's that are styled the same. Another option is to have multiple class names.
